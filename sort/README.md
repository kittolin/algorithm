## 线性排序
任何基于比较的排序算法的时间复杂度下限是 O(nlogn)，要想实现线性时间内的排序，需要对数字的范围加以限制。

采用计数排序，以空间换时间，时间复杂度 O(n + r) (r: 数字的范围)
